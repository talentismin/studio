iOS App

RunningHero

給 iPhone-only 跑者的進階訓練分析。不用買手錶、不用配硬體，跑完就有 VO2max / TRIMP / HRV / 燃脂底物等手錶級別指標。

主要功能

  • GPS 路線追蹤、配速、海拔、步頻
  • 藍牙心率帶連線（Polar H10 等 BLE HRM）
  • HR Zone（Karvonen 5 區）、HR Recovery、靜止心率歷史
  • VO2max（Daniels 公式）、TRIMP / ATL / CTL / TSB 訓練負荷
  • 燃脂底物推估（脂肪 vs 碳水比例）、有氧效率漂移
  • HRV 晨測（RMSSD），與訓練負荷交叉建議
  • 賽事倒數 + Daniels/Riegel 預估完賽時間，內建台灣賽事日曆
  • 間歇課表（structured workout） + 自動分段 + 語音教練
  • Apple Health 雙向同步、GPX / TCX 匯出（Strava / Garmin Connect）
  • 選配繁中 AI Coach（自帶 Anthropic API key）
